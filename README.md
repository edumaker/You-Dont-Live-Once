# 数字人激光标记系统

基于智谱 GLM-4V 视觉模型的激光笔标记识别与知识库系统。自动检测视频中的激光笔标记，AI 识别标记内容，生成问答并构建个人知识库。

## 功能特点

- **双色激光检测**：自动识别红色和绿色激光笔标记
- **AI 视觉理解**：调用智谱 GLM-4V 分析标记区域的内容
- **交互式问答**：AI 描述看到的内容，你回答为什么标记
- **知识库构建**：自动整理标记、回答、标签，支持回顾查询
- **多场景适用**：网课学习、文档阅读、代码演示等任何屏幕场景
